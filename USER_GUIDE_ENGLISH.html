4. Tournament Configuration

Before You Start

You must configure your tournament settings before entering any match data. This sets up the scoring system and tournament structure.

Accessing Configuration

  1. From the main menu, tap Configure Tournament
  2. You'll see the configuration screen

Configuration Options

1. Total Days

Enter the total number of days your tournament will run.

  • Example: If your tournament runs for 5 days, enter 5
  • Minimum: 1 day

2. Matches Per Day

Enter how many matches will be played each day.

  • Example: If you play 2 matches per day, enter 2
  • Minimum: 1 match

3. Points Per Kill

Set how many points each kill is worth.

  • Example: If each kill = 1 point, enter 1
  • Can be 0 or any positive number

4. Rank Points

Configure points awarded for each placement (rank 1 to 25).

Adding Rank Points

  1. Click â• Add Rank
  2. Enter the Rank Position (1-25)
  3. Enter the Points for that rank
  4. Repeat for all ranks you want to configure

ğŸ’¡ Default Scoring

If you don't specify points for a rank, it defaults to 0. You only need to configure ranks that award points!

Saving Configuration

  1. Review all your settings
  2. Tap ğŸ’¾ Save
  3. You'll see a success message: "âœ… Configuration saved successfully!"

âš ï¸ Changing Configuration Later

You can modify configuration at any time, but be aware:

  • Reducing "Matches Per Day" will hide data from higher-numbered matches
  • Reducing "Total Days" will hide data from later days
  • Data is NOT deleted, just hidden until you increase the values again

8. Troubleshooting

Common Issues & Solutions

Issue: "No configuration found" message

Solution: You must configure tournament settings before entering match data. Go to Main Menu â†’ Configure Tournament.

Issue: I switched matches and my data disappeared!

Solution: Your data is saved in temporary memory. Switch back to that match to see it again. Click Save to persist it permanently.

Issue: I closed the app and my data is gone!

Solution: Unsaved data is lost when the app closes. Always click the Save button before exiting.

Issue: Match boxes are all gray

Solution: No data has been entered yet. Click a match box to start entering data.

Issue: Access key shows "already used"

Solution: Each key can only be used once. Contact your tournament administrator for a new key.

Issue: Session expired / Auto-logged out

Solution: Your session duration has ended. You need a new access key to log in again. All tournament data is preserved and will be accessible after you log back in.

Issue: Can't save configuration

Solution: Make sure all required fields are filled:

  • Total Days (minimum 1)
  • Matches Per Day (minimum 1)
  • Points Per Kill (0 or more)

Issue: Leaderboard shows "No data"

Solution: No match results have been saved yet. Enter and save at least one match's data.

Data Safety

ğŸ’¡ Best Practices

  • Save frequently: Click Save after completing each day's matches
  • Double-check entries: Review data before saving
  • Use reset carefully: Reset operations are permanent
  • Check match colors: Green = saved, Yellow = unsaved

Need Help?

If you encounter issues not covered in this guide:

  1. Check if you've saved your configuration
  2. Verify all required fields are filled
  3. Try restarting the app
  4. Contact your tournament administrator
